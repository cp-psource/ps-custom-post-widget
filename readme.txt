Ermöglicht die Anzeige von benutzerdefinierten Beitragstypen und normalen Beiträgen mit Beitragsbildern und Auszügen als Widget.

== Description ==

Mit “Letzte benutzerdefinierte Beiträge” kannst Du eine Liste mit benutzerdefinierten Beiträgen in einem beliebigen Widget-Bereich Deiner Webseite anzeigen.
Es ist einfach, aktuelle Beiträge hinzuzufügen, aber was ist mit benutzerdefinierten Beitragstypen? 
Dieses Plugin fügt Deiner Seite ein Widget hinzu, mit dem aktuelle Beiträge von jedem Beitragstyp angezeigt werden.

Zeige Deine neuesten Produkte, Kleinanzeigen oder andere benutzerdefinierte Beitragstypen an.
